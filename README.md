# 🎓 Student Management System in C

## 📖 Overview
A menu-driven Student Management System developed in C to manage student records efficiently using structures and file handling.

## ✨ Features
- Add Student Record
- Delete Student Record
- Update Student Record
- Search Student Record
- Display All Records
- Permanent Data Storage using Files

## 🛠️ Technologies Used
- C Programming
- Structures
- File Handling
- Functions
- Menu-Driven Programming

## 🚀 How to Run
1. Compile the program:
   gcc student_management.c -o sms

2. Run the executable:
   ./sms
